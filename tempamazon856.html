Online Programming Round: (5 methods, 2 hours)

1)\xc2\xa0\xc2\xa0\xc2\xa0\xc2\xa0 A sentence is given which contains lowercase English letters and spaces. It may contain multiple spaces. Get first letter of every word and return the result as a string. The result should not contain any space. Complete the following method:

static String getFirstLetterWord(String text) {\xc2\xa0 }

2)\xc2\xa0\xc2\xa0\xc2\xa0\xc2\xa0 Given an array. Iterate it for the given number of times. And then return the summation of the resultant elements.

Ex: Array is { 1,2,5,6}, N=2

After 1st iteration: {2-1, 5-2, 6-5}={1,3,1}

After 2nd : {3-1, 1-3}={2,-2}

Sum is 2\xc2\xa0 + (-2) = 0

If only one element remains in the array, the element remains the same after applying the iteration. Complete the method.

static int iterateSequence(Vector<Integer> a, int N) { }

3)\xc2\xa0\xc2\xa0\xc2\xa0\xc2\xa0 Find Nth largest element in the BST. Complete the method.

staticintnLargeBST(Node root, int N) {}

Given that

\r\nclass Node\r\n{\r\n    Node left, right;\r\n    int data;\r\n    Node(intnewData)\r\n    {\r\n        left = right = null;\r\n        data = newData;\r\n    }\r\n}

4)\xc2\xa0\xc2\xa0\xc2\xa0\xc2\xa0 Swap adjacent nodes in the linked list. Change the links, not the data. Complete the method.

Ex:1, 2, 3, 4

ex: 1,2,3,4,5

op: 2, 1, 4, 3, 5
