Amazon Interview Experience | 193 (For SDE-1)

  • Difficulty Level :\nMedium
  • Last Updated :\n03 Jun, 2021

I had interview with amazon.com for SDE1 position.\xc2\xa0

1st Round \xe2\x80\x93 (Written Round)-\xc2\xa0
25 Multiple choice questions(Networking, OS, DBMS, C concepts)\xc2\xa0
2 coding questions were also asked, for which paper code was required.\xc2\xa0

2nd round \xe2\x80\x93\xc2\xa0
1. Placing N chess queens on an N\xc3\x97N chessboard so that no two queens attack each other. For example, following is a solution for 4 Queen problem.\xc2\xa0

I had a long discussion on this question finally he was satisfied with my solution.\xc2\xa0

2. Write a program to find the sum of maximum sum subsequence of the given array such that the integers in the subsequence are sorted in increasing order.\xc2\xa0

3. Given a Binary Tree, check if all leaves are at same level or not.\xc2\xa0

3rd round \xe2\x80\x93\xc2\xa0
Given a sorted array arr[] and a number x, write a function that counts the occurrences of x in arr[]. Expected time complexity is O(Logn)\xc2\xa0

Find k largest elements in an array. Elements in array can be in any order.\xc2\xa0

4th round(Hiring manager)-\xc2\xa0
1. Questions around current job, challenges faced. Any instance of conflict and how you resolved it.\xc2\xa0
2. A lot of discussion on projects and work experience.\xc2\xa0
3. A large file is given you have to find all the anagrams. Told him to divide the file in chunks for parallel processing and then check anagrams using heap.\xc2\xa0

5th round(Bar Raiser)-\xc2\xa0
1. Why you are leaving your current company so early?\xc2\xa0
2. An integer is given to you,first convert it into binary then reverse this binary string and print decimal corresponding to this reversed string.\xc2\xa0
3. He asked one question related to REALLOC function.\xc2\xa0

In every round, after each question, complexity of the algorithm was asked. Its good to ask questions to get more clarification about the question.\xc2\xa0

I want to thank the entire team of geeksforgeeks. It is a great portal and it helped me a lot in preparing for Amazon interview.\xc2\xa0

Tips: Discuss various approaches with interviewer. Try to interact continuously with interviewer. they were very friendly during interview.\xc2\xa0
