Amazon Interview | Set 23

  • Last Updated :\n14 Jun, 2019

I was recently interviewed for SDE1 position in Amazon Bangalore. 1 online coding test followed by 5 rounds of F2F onsite interviews.

Online Coding Test (4 Problems, 2 hours)\xc2\xa0From Interviewstreet
All below problems had multiple test cases for which the code was validated against.
1.\xc2\xa0Code for converting floating point decimal number to binary numbers. If the number cannot be converted, state so.
2.\xc2\xa0Given an integer array A of size n. Given an integer k < n. Construct an array B, such that
B[i] = min{A[i], A[i+1], A[i+2], A[i+3], \xe2\x80\xa6\xe2\x80\xa6., A[i+k]}
Solve in time complexity better than O(nk).
Hint: use min Heaps
3.\xc2\xa0A singly liked list. Can have a loop. Detect it and find the size of list.
4.\xc2\xa0A singly link list and a number \xe2\x80\x98K\xe2\x80\x99, swap the Kth node from the start with the Kth node from the last. Check all the edge cases.
Sample Input: 1->2->3->4->5->6->7->8 and K = 3
Sample Output : 1->2->6->4->5->3->7->8

Sample Input: 1->2->3->4->5->6->7->8 and K = 10
Sample Output: print error \xe2\x80\x9cLIST IS OF LESSER SIZE\xe2\x80\x9d.

Face to face: Round 1(Technical, 1 hour):
1.\xc2\xa0Given a K sorted array. Sort it with minimum time complexity.
O(nlogk) solution was expected. Code was required.
2.\xc2\xa0Given a file with many product name of an company. You have to find out unique name in the file.\xc2\xa0Suppose\xc2\xa0mobile,laptop,notepad,desktop,pen,mobile,pen .. etc is given we have to print laptop,notepad,desktop. Pen and mobile should be remove due to\xc2\xa0duplicity.\xc2\xa0

Code was required.I gave o(n) time complexity solution for it using 1 hash table and Doubly Link List.\xc2\xa0


Face to face: Round 2(Technical, 1 hour):
1.\xc2\xa0Given a Sorted array with one\xc2\xa0missing\xc2\xa0number. I have to find first\xc2\xa0missing\xc2\xa0number.Code was required.\xc2\xa0I gave solution with o(logn) time complexity.

2.\xc2\xa0Give a Building with n floor. A person can take 1 step or 2 step to climb. Find the number of ways to reach nth floor.\xc2\xa0Code was required


Face to face: Round 3(Technical, 1 hour 20 minutes):
1.\xc2\xa0Given an Sorted Array with duplicates I have to find first index of any duplicates. Suppose 12222333355578999 first Index of 2 in 1. Code was required.I gave O(logn) Solution.

2. Given an binary tree. Traverse it in zig-zag manner. Code was required.Solved using a 2 stack.

3. In a snake ladder game without snake and ladder :). If some one is playing then we have to find\xc2\xa0probability to win the game of any player. Condition of winning is if you are on 96 and 5 comes in dice then you loose the game and If you are at 96 and 4 comes then only you will win the game. But you cant use dice more than Y time.

I gave o(XY) Solution through DP. Where X is sum.

Interviewer was very happy after that.
