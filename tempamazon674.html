Amazon Interview Experience | Set 184 (Off Campus for SDE1)

  • Difficulty Level :\nMedium
  • Last Updated :\n20 May, 2021

I interviewed with Amazon, Hyderabad. Interview was in October, 2014. I was 4 months experienced, and was applying for SDE-1 position.\xc2\xa0

I had 4 interviews, and my experiences are as follows.\xc2\xa0

1st Round\xc2\xa0
Q1 \xe2\x80\x93 Clone a graph. ( Similar Question -> https://www.geeksforgeeks.org/clone-linked-list-next-arbit-pointer-set-2/)\xc2\xa0
Basically while cloning, when you create a new node in the cloned graph, have a hashmap which maps the old node to new node.\xc2\xa0
So in the hashmap key = old node, value = new node.\xc2\xa0

Q2 \xe2\x80\x93 Given an array of numbers, for each number print the first number to it\xe2\x80\x99s left which is greater than the current number.\xc2\xa0

    Example\n        Input -> 5,3,2,4,8,6\n        Output-> -1, 5,3,5,-1,8\n    Use stacks. Simple O(n) solution.

2nd Round\xc2\xa0
Q1 \xe2\x80\x93 Find the row number according to the excel nomenclature \xe2\x80\x93> (the opposite of this, https://www.geeksforgeeks.org/find-excel-column-name-given-number/)\xc2\xa0

    i.e given Z, Output -> 26\n    Given AX, Output -> 50

Q2 \xe2\x80\x93 Find the number of islands in a 2d Array -> https://www.geeksforgeeks.org/find-number-of-islands/\xc2\xa0

3rd Round(Hiring Manager)\xc2\xa0
Q1 \xe2\x80\x93 Given a number in words, print the number.\xc2\xa0

    Example 1) Input - "One Hundred and Five", Output should be "105" \n    Example 2) Input - "One Thousand and two hundred", Output should be "1200"\n    Example 3) Input - "Twelve hundred", Output should be "1200"\n    Example 4) Input - "Five Lacs", Output should be "500000"\n    Example 5) Input - "Five Hundred Thousand", Output should be "500000"

There were a lot of edge cases, and discussion with regards to this. ( I took a lot of time and the manager did not seem to pleased. \xf0\x9f\x99\x81 )\xc2\xa0

Q2- In a sorted array, find two numbers who\xe2\x80\x99s difference is k.\xc2\xa0

Given -> 1,2, 3,4,7,8,9,11 and k=7\nOutput -> 1,8 or 2,9, or 4,11 (Print any one)

https://www.geeksforgeeks.org/find-a-pair-with-the-given-difference/\xc2\xa0

A lot of questions as to Why Amazon, Why do you want to leave so early.\xc2\xa0

4th Round(Bar Raiser)\xc2\xa0
Q1 \xe2\x80\x93 Given a sorted array find a number. (Simple Binary Search)\xc2\xa0
Now consider repetitions and find the left most occurrence. (Binary Search to first find number, then again Binary Search to find left most occurrence)\xc2\xa0

Similarly find the right most occurrence. (Binary Search to first find number, then again Binary Search to find right most occurrence)\xc2\xa0

Now using the above two functions, find the number of times the element is present.\xc2\xa0
Note \xe2\x80\x93 Cases like where number is NOT present,\xc2\xa0

Q2- Assembly Line Scheduling. (Very Simple DP.)\xc2\xa0
https://www.geeksforgeeks.org/dynamic-programming-set-34-assembly-line-scheduling/\xc2\xa0

Result\xc2\xa0
After three weeks, I got a generic email saying that I could not clear it. I had a really irresponsible recruiter who did not have the courtesy to give me a call and just convey some feedback. Eventually, I think I took too much time with the first question in the manager round.\xc2\xa0
