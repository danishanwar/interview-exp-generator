Round 1 (Online Test): Online test was conducted on Mettl platform. The duration was for 1hr 30 min. It was webcam proctored and consisted of 28 MCQs and 2 coding questions. MCQs were based on data structures, predict output, C++, and algorithms.\xc2\xa0

Round 2: The interview started with \xe2\x80\x9ctell me about yourself\xe2\x80\x9d. After that, he asked coding questions.

  1. Find the longest substring containing distinct characters. \xc2\xa0

    Ex:

    Input: abbabcdd\r\nOutput: 4

    Reference:https://www.geeksforgeeks.org/length-of-the-longest-substring-without-repeating-characters/

  2. Given preorder and pre_LN that contains whether the node is leaf nodes or not, \xc2\xa0construct a binary tree. \xc2\xa0

    Ex:\xc2\xa0

    pre =  {1,2,45,3,5}  pre_LN={N, N, L, L, L}\r\nTree:\r\n       1\r\n     /   \\\r\n    2     5\r\n  /   \\\r\n45      3\r\nThe interviewer was very keen on edge case handling.  \r\nFor example, Function  should return null  if:\r\npre={1,2,3}   pre_LN={L, N, N}

    Reference: https://www.geeksforgeeks.org/construct-a-special-tree-from-given-preorder-traversal/

The duration was for 1 hour.

Round 3: The interview started with \xe2\x80\x9ctell me about yourself \xe2\x80\x9c.

  1. A bag contains magic balls. \xc2\xa0Each ball has a specific weight. \xc2\xa0Every time, balls with two highest weight are removed (let the balls be X and Y) and collision which may or may not result in a new ball following the given constraints :

    • Case 1: If X>Y then they result in a new ball of weight X-Y
    • Case 2: If Y>X they result in a new ball of weight Y-X
    • Case 3: X=Y No new ball
    • Example: [2,4,6,8]
    • Pick 6,8 \xe2\x80\x93 > New ball=2 -> Array becomes [2,4,2]
    • Pick 4,2 -> New ball=2-> Array becomes [2,2]
    • Pick 2,2 -> Array is empty -> return 0
    • Example2: \xc2\xa0[3,10,17]
    • Pick 10,17 \xe2\x80\x93 > New ball=7 -> Array becomes [3,7]
    • Pick 3,7 -> New ball=4-> Array becomes [4]

    Since the array contains only 1 ball, return its weight -> return 4

  2. There are n workers, each having some capacity. The workers are given rotis based on their capacity, \xc2\xa0i.e workers with a higher rating will get a more number of rotis. A worker can only know the rotis and capacity of two of his neighbors, one on the left and other on the right. Given an array specifying the capacity of workers, find the minimum rotis that should be given for each worker, \xc2\xa0so that no worker feels unfair.

    Example:\xc2\xa0

    Input: 1 3 5 4 (ratings for 4 workers)\r\nOutput: 1+2+3+2 = 7

    Example:\xc2\xa0

    Input: 5 3 4 2 1 6\r\nOutput: 2+1+3+2+1+2 = 11.

  3. Since there was time left, \xc2\xa0he asked ACID properties in DBMS and mutex, semaphores, deadlocks (operating system concepts).

The duration was for 1 hour.

Result: Selected
