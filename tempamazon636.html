Amazon visits our campus for recruitment. Here I am sharing my experience of the whole process.\xc2\xa0

Round 1:\xc2\xa0

First there was an online round on hackerrank and consisted of 20 mcqs and two coding questions.\xc2\xa0

mcqs were mainly from c/c++ input/output DS, OS .Here are those two coding questions:\xc2\xa0

\xc2\xa0\xc2\xa0

Ques1: An array containing heights of building was given. Its a rainy season. Calculate the amount of water collected between all the buildings.\xc2\xa0

Input: 1 5 3 7 4 2\xc2\xa0

Output: 2\xc2\xa0

Ques2: Two strings are given check if anagram of one string is contained in other string.\xc2\xa0

\xc2\xa0\xc2\xa0

Round 2(F2F 1):\xc2\xa0

Three coding questions were asked, all from geeks.\xc2\xa0

Ques1: In a party of N people, only one person is known to everyone. Such a person\xc2\xa0may\xc2\xa0be present\xc2\xa0in the party, if yes, (s)he doesn\xe2\x80\x99t know anyone in the party.\xc2\xa0We can only ask questions like \xe2\x80\x9cdoes A know B?\xc2\xa0\xe2\x80\x9c. Find the celebrity.\xc2\xa0

https://www.geeksforgeeks.org/the-celebrity-problem/\xc2\xa0

Twist: this time celebrity may or may not know some persons in the party. Suggested him O(n^2) approach. Using stack we can minimize comparisons.\xc2\xa0

Ques2: Given an unsorted array that contains even number of occurrences for all numbers except two numbers. Find the two numbers which have odd occurrences in O(n) time complexity and O(1) extra space. Told him that I know its approach already. He asked me for the approach.\xc2\xa0

https://www.geeksforgeeks.org/find-the-two-numbers-with-odd-occurences-in-an-unsorted-array/\xc2\xa0

Ques3: An array of size n-1 is given contains numbers from 0 to n. Find the two missing numbers in O(n) time and O(1) space. Suggested him method given in :\xc2\xa0

https://www.geeksforgeeks.org/find-the-smallest-positive-number-missing-from-an-unsorted-array/\xc2\xa0

\xc2\xa0\xc2\xa0

Round 3(F2F 2):\xc2\xa0

He starts with my introduction and some questions on classification as in my machine learning project I have made a classifier.\xc2\xa0

Ques 1: Find next Greater element for each element of the given array.\xc2\xa0

https://www.geeksforgeeks.org/next-greater-element/\xc2\xa0

Ques 2: Find if a binary tree is BST or not. Suggested method 3 given in this post.\xc2\xa0

https://www.geeksforgeeks.org/a-program-to-check-if-a-binary-tree-is-bst-or-not/\xc2\xa0

Ques 3: Given a matrix calculate minimum cost path to reach from top-left to bottom-right. Suggested him the dp approach then he asked me if this will work with negative edges or not. I said yes it would work.\xc2\xa0

He then twisted the question a bit\xe2\x80\xa6now one can move in either of the four direction left,top,right and bottom and can contain negative edges. Suggested him recursive approach having exponential time complexity. Asked me in which case it won\xe2\x80\x99t work. After sometime told him if it contains negative cycle. I think it can easily be done with graphs.\xc2\xa0

Ques 4: coins are lined up (eg. 100 200 50 2) Two players playing this game can pick a coin from either of the left or right end. Player with maximum sum will win. Both players are optimally playing the game. You have to tell that whatturn(1 or 2) should be chosen in order to win. Like in above example if you play first then can select coin 2(you want to get that 200 coin to win).\xc2\xa0

\xc2\xa0\xc2\xa0

Round 4(F2F 3):\xc2\xa0

He started will \xe2\x80\x9ctell me about yourself\xe2\x80\x9d and then he asked me about \xe2\x80\x9cVirtualization\xe2\x80\x9d one of my project. A long discussion on the same. Questions like what happens if he wants to have a trace of his previous work and other project related questions.\xc2\xa0

Ques 1: Sort a 2GB file line by line. Avoid use of extra space (or use minimum space).\xc2\xa0

Suggested him solution with trie but he said it would take space if there are no common prefixes. Then i gave him a solution with hashing(store hash value per line and sort it) but he was not satisfied with time complexity) then finally suggested counting sort. As we can have only 256 different characters at max in any line. Time complexity was O(n*m) n no of lines and m no of characters per line. Satisfied =D.\xc2\xa0

Ques 2: He gave me a small code to debug. Below code is having four bugs.\xc2\xa0

Unsigned inti;\xc2\xa0

For(i=100,i<=0,\xe2\x80\x93i)\xc2\xa0

Printf(\xe2\x80\x9c%d\\n\xe2\x80\x9d,i);\xc2\xa0

Ques 3: What happens when you hit your college url. Describe all steps. Then he asked some questions on networks like http, tcp, port, switches etc.\xc2\xa0

Ques 4: Print all permutations of a string. Asked me for the approach and code it.\xc2\xa0

Round 5(F2F 4):\xc2\xa0

Long discussion on my summer internship project. Project was on \xe2\x80\x9cMachine Translation\xe2\x80\x9d (translation of one language document into other language). Mine was converting English to Hindi. Asked me for the approach and discussion on the same.\xc2\xa0

Then he gave me this problem.\xc2\xa0

There are number of clients requesting pages of a service. Find number of pages requested by clients in each 5 minutes. Gave a solution with queue using linked list and map.\xc2\xa0

Then a long discussion on my minor project which was on machine learning. The interviewer was having deep knowledge on machine learning So he asked me a lot on the same. Then he asked me if I have studied it as a subject. I said no. He said \xe2\x80\x9cthat is what I wanted to know\xe2\x80\x9d. He was impressed with my projects.\xc2\xa0
