Telephonic Interview 1. (Telephonic + Google Docs Shared)

Q1. Given an array of integers, you need to find the local maxima.

Example : [1 3 5 4 7 10 6]\r\n\r\nOutput: 5 or 10\r\n\r\nExplanation: Any of the local maxima can be the output. \r\nHere 5 is greater than 3 and 4, 10 is greater than 7 and 6. \r\n

Hint : Refer this article.

Q2. Given a sequence of brackets, how will you identify if its valid or not.

Example : ({[][]})\r\n\r\nOutput: Valid\r\n\r\nExplanation: Every opening bracket has a closing bracket.\r\n\r\nExample: ({[]]}) \r\n\r\nOutput: Invalid \r\n\r\nExplanation : Every opening bracket does not have a closing bracket.\r\n

Q1. There are 2 arrays. Smaller is of size m and has m elements in sorted order. The bigger array is

positions are empty in the bigger array. Insert smaller array\xe2\x80\x99s m elements in m + n array has all numbers in sorted order.

Q2. Given a binary tree with integer values, find the sub-path with the maximum value in it

   \r\nExample : \r\n       1\r\n     / \xc2\xa0\xc2\xa0\\ \r\n    2 \xc2\xa0 \xc2\xa0\xc2\xa0 3\r\n  / \xc2\xa0 \\ \xc2\xa0/\xc2\xa0\xc2\xa0 \\\r\n 4\xc2\xa0\xc2\xa0\xc2\xa0\xc2\xa0 5 6\xc2\xa0\xc2\xa0\xc2\xa0 7\r\n\r\nOutput : Max path is 5, 2, 1, 3, 7\r\nExplanation : 5+2+1+3+7=18 is the maximum value that can spanned.\r\n\r\n      -100\r\n     /     \\\r\n    2        3\r\n   /  \\     /  \\\r\n  4    5   6    7\r\n\r\nOutput : Max path is 6, 3, 7\r\nExplanation : 6+3+7=16 is the maximum value that can spanned.\r\n
