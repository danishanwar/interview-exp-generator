Round 1:\xc2\xa0

Question: A boolean expression is given in the form of a string. It contains one variable x; logical operators \xe2\x80\x93 \xe2\x80\x98and\xe2\x80\x99, \xe2\x80\x98or\xe2\x80\x99 and relational operators \xe2\x80\x93 \xe2\x80\x98>\xe2\x80\x99, \xe2\x80\x98<\xe2\x80\x98 (there is no >= or <=). Find if the expression always evaluates to False. If yes, output False, otherwise if there exists at least one x such that the given expression can be true, output true.

Example:

1. \xe2\x80\x9cx<0 and x>5\xe2\x80\x9d

Output \xe2\x80\x93 False

Explanation \xe2\x80\x93 This can never be true as there is no \xe2\x80\x98x\xe2\x80\x99 such as x<0 and x>5. So, the given boolean expression always evaluates to false.

2. \xe2\x80\x9cx>0 or x<-1\xe2\x80\x9d

Output \xe2\x80\x93 True

Explanation \xe2\x80\x93 We have at least one \xe2\x80\x98x\xe2\x80\x99 for which given boolean expression evaluates to true. For example, put x=2 in the given expression, and it evaluates to true.

Hint:\xc2\xa0Whenever there is only \xe2\x80\x98or\xe2\x80\x99 in the boolean expression, the result is always true. (Eg: x>0 or x<0 \xe2\x80\x93 There exists some x such that this is true and whatever be the latter part of the expression, it evaluates to true as only \xe2\x80\x98or\xe2\x80\x99 is present. If there is no \xe2\x80\x98or\xe2\x80\x99 present (only \xe2\x80\x98and\xe2\x80\x99 is there), then we check for the expressions \xe2\x80\x93 if you find at least two contradicting expressions as in example 1 (that is their solution sets are disjoint), then the output is False (as we have only \xe2\x80\x98and\xe2\x80\x99 logical operation which evaluates to False unless all the expressions are True), otherwise it is True.

I have no idea how to approach the problem when both \xe2\x80\x98and, \xe2\x80\x98 \xe2\x80\x98or\xe2\x80\x99 are present in the expression, and I could not find such a problem anywhere on the internet.
