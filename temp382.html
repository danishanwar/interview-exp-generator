Section 1:\xc2\xa0
The first section is about creating a DOM element which we name as \xe2\x80\x98script\xe2\x80\x99 and then set the \xe2\x80\x98type\xe2\x80\x99 and \xe2\x80\x98src\xe2\x80\x99 of this element using the DOM setAttribute() method which adds the specific attribute to an element, and gives it the specified value. The second code section contains two arrays \xe2\x80\x98nameArr\xe2\x80\x99 and \xe2\x80\x99emailArr\xe2\x80\x99 which contains several names and emails which we can iterate over and randomly choose values from. In this section we are making use of Math.floor() and Math.random() methods provided by the Math Object in javascript. Then we make use of Math.floor() method which is to round off. Then we assign the values obtained to a variable which we will later pass to the DOM elements which will ultimately get inserted into the form.\xc2\xa0
Section 4:\xc2\xa0
In the last code section, we are basically assigning values to the DOM elements that we extract using the JQuery \xe2\x80\x98$\xe2\x80\x99 sign which demands a selector inside it and then it performs some action on it.
